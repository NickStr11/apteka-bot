# Model Selection

| Задача | Модель |
|--------|--------|
| Архитектура, сложные баги | Opus / o1-pro |
| Быстрая разработка, CRUD | Sonnet / GPT-4 |
| Фронтенд | Sonnet / GPT-4o |
| Legacy код | Opus / o1 |

**Стратегия:**
1. Начни с Sonnet
2. Переключись на Opus если не работает после 2-3 попыток
3. Вернись на Sonnet для полировки
